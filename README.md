An easy way to generate projects for sdev.

## Guide

- <a href="#installation">Installation</a>
- <a href="#usage">Usage</a>

## Installation

```bash
npm i -g scli.js
# yarn add -g scli.js
# pnpm add -g scli.js
```

## Usage
Execute the following command in your terminal to open a generation menu:
```bash
sdev menu
```
